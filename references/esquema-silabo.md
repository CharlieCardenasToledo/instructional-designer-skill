# Contrato del sílabo canónico

El `README.md` de la raíz del curso es la fuente de verdad. Validar este contrato antes de generar una guía.

## Encabezado requerido

```markdown
# CÓDIGO — Nombre de la asignatura

**Asignatura:** CÓDIGO — Nombre de la asignatura
**Periodo académico ordinario:** Abril–Agosto 2026
**Créditos:** 3
**Semestre:** Tercero

## Descripción del curso

Descripción breve.
```

## Semana requerida

```markdown
### Semana 01 — Tema central

**Unidad:** Unidad 1 — Nombre

**Tema / contenido semanal:**
- Tema principal
- Segundo tema

**Resultado de aprendizaje:**
- Docencia: ...
- Práctica: ...
- Autónomo: ...

**Herramienta de aprendizaje:**
- Autor (Año). Título. Capítulo o sección.
- Recurso o herramienta.

**Horas:** Docencia: 2 | Práctica: 1 | Autónomo: 3

**Actividades calificadas:**
- AC-01 — Nombre — 10 puntos
```

## Reglas

- Usar `**Resultado de aprendizaje:**` en singular.
- Conservar los componentes Docencia, Práctica y Autónomo cuando existan.
- Usar `**Herramienta de aprendizaje:**` para bibliografía y recursos.
- Usar `**Actividades calificadas:**` en plural. Escribir `- Ninguna` si no aplica.
- Usar números de semana con dos dígitos.
- Separar semanas con `---`.
- No sobrescribir un sílabo existente sin crear una copia `.bak-YYYYMMDD-HHMMSS`.
